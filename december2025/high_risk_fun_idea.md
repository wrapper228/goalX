## Что нам понадобится
1.  **Жертвенная среда:** Виртуальная машина (VMware/VirtualBox) с Windows или "чистый" ноутбук. *Не запускать на основной машине с важными данными!*
2.  **API Key:** Anthropic API Key с доступом к Claude 3.5 Sonnet (именно он умеет в Computer Use). Нужен баланс ~$10-20.
3.  **Docker:** Самый простой способ запустить их эталонную реализацию.

---

## План действий

### 1. Подготовка среды (30 мин)
*   Установить Docker Desktop на жертвенную машину.
*   Получить ключ API в консоли Anthropic.

### 2. Запуск "Официального Гомункула" (15 мин)
Anthropic выложили готовый референс-контейнер. Не надо писать код с нуля, надо просто запустить их демо.

```bash
export ANTHROPIC_API_KEY=sk-ant-...

docker run \
    -e ANTHROPIC_API_KEY=$ANTHROPIC_API_KEY \
    -v $HOME/.anthropic:/home/computeruse/.anthropic \
    -p 8080:8080 \
    -p 8501:8501 \
    -it ghcr.io/anthropic/anthropic-quickstarts:computer-use-demo-latest
```

*(Команда может чуть отличаться для Windows PowerShell, суть та же)*.

### 3. Первый контакт (Fun Part)
*   Открыть `http://localhost:8080` в браузере.
*   Там будет интерфейс: Чат + Окно виртуального рабочего стола (Streamlit).
*   **Задача 1 (Тест):** "Открой Calculator и посчитай 42 * 153".
    *   *Смотрим, как он двигает мышкой и нажимает кнопки.*

### 4. Повышаем ставки (Real World Task)
Если калькулятор сработал, даем "Злую Задачу" (Wicked Problem light):
*   **Промпт:** "Зайди на `news.ycombinator.com`, найди топ-статью про AI, перейди на неё, скопируй первый абзац текста, открой Notepad и сохрани этот текст в файл `hacker_news_ai.txt` на рабочем столе".

### 5. Анализ (The Aftermath)
Когда он закончит (или упадет), задать себе вопросы для `gap_analysis.md`:
1.  Насколько это было медленно? (Лаг между "подумал" и "кликнул").
2.  Сколько денег это съело? (В консоли будет статистика токенов — там передаются скриншоты, это дорого).
3.  Где он затупил? (Не нашел иконку? Сайт долго грузился?).
4.  **Чувствую ли я "Силу"?** Или я чувствую "Испанский стыд за робота"?
