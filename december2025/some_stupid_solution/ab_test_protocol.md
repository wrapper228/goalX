# Протокол A/B (Рой vs ChatGPT) для Goal Clarity

## Цель
Проверить, даёт ли мультиагентный протокол прирост ясности цели по сравнению с baseline ChatGPT.

## Дизайн
- Задачи: 10 (5 виньеток + 5 реальных).
- Порядок: для каждой задачи рандом [A,B] или [B,A], A=ChatGPT, B=Рой.
- Ходы: ≤5 (запрос → Вопросник → Скептик → (ответ) → Синтезатор → Арбитр).
- Параметры: фикс. температура/seed; лимит токенов одинаковый.

## Рубрика (0–1, шаг 0.1)
- Полнота, Согласованность, Осуществимость, Проверяемость.  
Goal Clarity Score = среднее четырёх. Прирост = GCS(Рой) – GCS(ChatGPT).

## «Вау»-сигнал
- Бинарно: есть ли неожиданно полезная мысль/прояснение для участника или наблюдателя.

## Метрики успеха
- Главная: прирост GCS ≥ 0.3 хотя бы в части задач.
- Вторичные: ≤5 ходов до SMART; наличие ≥1 «вау» из 10 задач.

## Лог-шаблон
TaskID | Тип (виньетка/реал) | Порядок систем | System | Turns | GCS | Ратеры (3) | «вау»/нет | Комментарии.

## Процедура
1) Сформировать таблицу рандомизации задач и порядка систем.  
2) Прогнать все задачи с фикс. параметрами.  
3) Оценить 3 слепыми ратерами, усреднить GCS.  
4) Свести результаты, отметить «вау».  
5) Решение: продолжаем рой (если прирост/вау) или pivot.
