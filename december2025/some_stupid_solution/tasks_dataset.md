# Шаблон задач (виньетки + реальные)

## Виньетки (5 шт.)
1) Карьера: «Хочу сменить работу, боюсь потерять доход» — скрыто: роль/рынок/дедлайн/допустимая просадка.
2) Учёба: «Думаю о PhD/магистратуре за рубежом» — скрыто: зачем (ресурсы/статус/среда), бюджет, сроки.
3) Проект: «Хочу pet-проект с ИИ» — скрыто: цель (портфолио/доход/исследование), объём, дедлайн.
4) Время/отношения: «Хочу больше времени на семью и спорт» — скрыто: расписание, отказ от задач 3-го уровня.
5) Доход: «Хочу увеличить доход» — скрыто: способ (карьера/фриланс/продукт), риск/время.

## Реальные (шаблоны — заполни своими)
6) Твой текущий карьерный шаг (LLM/recsys/инфра?).
7) Академический трек (публикация/поступление).
8) Конкретный pet-проект (goal-elicitation/рой/UX).
9) Личное расписание (спорт/семья/учёба/работа).
10) Финансовая цель (сумма, горизонт, способ).
