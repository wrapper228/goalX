# Сценарий демо-видео (3–4 минуты)

## 0:00–0:20 — Зачем
- Проблема: LLM уточняют факты, но не строят когнитивную карту пользователя → слабая ясность цели.
- Цель: показать, как мультиагентный протокол повышает Goal Clarity.

## 0:20–1:00 — Протокол
- Роли: Вопросник, Скептик, Синтезатор, Арбитр.
- Ходы (≤5): запрос → Вопросник → Скептик → (ответ) → Синтезатор → Арбитр.
- Метрика: Goal Clarity (полнота, согласованность, осуществимость, проверяемость, 0–1).

## 1:00–2:30 — Живой кейс (пример «смена работы»)
- Покажи запрос пользователя.
- Ход Вопросника (2–3 вопроса), ход Скептика (риски/тест), короткий ответ пользователя, ход Синтезатора (SMART+3 шага+критерий), финал Арбитра.
- Выведи GCS для Роя и baseline ChatGPT (например 0.8 vs 0.6, прирост +0.2).
- Отметь «вау»-момент (критерий оффера ≥80% дохода).

## 2:30–3:10 — Результаты мини-A/B
- 3 кейса: смена работы, pet-проект, время/семья/спорт.
- Диапазон прироста GCS: +0.2…+0.25.
- «Вау» в 1–2 кейсах.

## 3:10–3:40 — Как повторить/контрибьютить
- Ссылка на репо: промпты, протокол, рубрика, датасет, логи.
- Призыв: добавляйте кейсы, роли, оценки; делайте свои A/B.

## 3:40–4:00 — Финал
- Миссия: строим архитектуры коллективного разума, где люди+ИИ быстрее находят цели и действуют.
- Контакты/ссылки.
