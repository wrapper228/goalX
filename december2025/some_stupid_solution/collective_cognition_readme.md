# collective-cognition-lab (скелет)

Открытые протоколы и демо для выявления скрытых целей через мультиагентный диалог (Вопросник, Скептик, Синтезатор, Арбитр).

## Быстрый старт
- Роли:  
  - Вопросник — задаёт 2–3 точных вопроса о целях/ограничениях/метриках.  
  - Скептик — атакует предпосылки, указывает риски, предлагает быстрый тест реальности.  
  - Синтезатор — формирует SMART-цель, 3 шага, критерий успеха.  
  - Арбитр — следит за лимитами, даёт финал (цель, шаги, критерий).
- Протокол: ≤5 ходов — запрос → Вопросник → Скептик → (ответ пользователя) → Синтезатор → Арбитр.
- Рубрика (0–1): полнота; согласованность; осуществимость; проверяемость. Goal Clarity = среднее.

## Метрики
- Главная: прирост Goal Clarity vs baseline (vanilla ChatGPT).  
- Вторичные: «вау»-сигнал (участник/ты удивлены пользой); шаги до SMART-цели; время/ходы до цели.

## Датасет задач
- 5 виньеток (карьера, учёба, pet-проект, отношения/время, деньги).
- 5 реальных задач (добавь свои текущие).

## Логи
- Шаблон: TaskID | Тип (виньетка/реал) | System (Рой/ChatGPT) | Turns | GCS | Ратеры (3) | «вау»/нет | Комментарии.
